# Complete

Configuration in this directory to create a polardb MySQL NormalMultimaster cluster service under Alibaba Cloud VPC and VSwitch.

## Usage

To run this example you need to execute:

```bash
$ terraform init
$ terraform plan
$ terraform apply
```

Note that this example may create resources which cost money. Run `terraform destroy` when you don't need these resources.
